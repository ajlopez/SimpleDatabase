# SimpleDatabase

Simple NoSQL Database implemented in JavaScript, Node.js, in-memory and file persistence, WIP.

## Installation

Via npm on Node:

```
npm install simpledatabase
```


## Usage

Reference in your program:

```js
var simpledatabase = require('simpledatabase');
```
