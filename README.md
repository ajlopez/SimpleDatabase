# SimpleDatabase

Simple NoSQL Database implemented in JavaScript, Node.js, WIP. Inspired by [RethinkDB](http://www.rethinkdb.com).

## Installation

Via npm on Node:

```
npm install simpledatabase
```

## Usage

Reference in your program:

```js
var simpledatabase = require('simpledatabase');
```
